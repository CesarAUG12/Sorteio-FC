# ⚽🎮 Sorteador de Times

**Sorteador de Times** é uma aplicação simples em **HTML, CSS e JavaScript** criada para organizar partidas de futebol com amigos ou campeonatos de FIFA.  
O projeto permite inserir jogadores e gerar equipes de forma **100% aleatória e equilibrada**, com duas opções de sorteio:

- Divisão por **número de equipes**  
- Divisão por **quantidade de jogadores por equipe**

👉 Acesse online: [Sorteio FC](https://sorteio-fc.netlify.app/)

---

## 🚀 Funcionalidades
- 📋 Inserir lista de jogadores manualmente.  
- 🔀 Sorteio aleatório e justo dos times.  
- ⚙️ Escolha entre dividir por número de equipes ou por jogadores por equipe.  
- 🎮 Seções independentes:
  - **Futebol Real** → ideal para organizar a pelada com os amigos.  
  - **FIFA/Game** → cada jogador recebe um time aleatório para campeonatos no videogame.  
- 📱 Interface **responsiva**, pensada para uso em dispositivos móveis.  
- 🎨 Layout simples e estilizado, com botões intuitivos e resultados exibidos em cards.  

---

## 📂 Estrutura do Projeto
- `index.html` → página inicial com escolha entre Futebol Real ou FIFA/Game.  
- `fut.html` → sorteador para partidas de futebol real.  
- `game.html` → sorteador para campeonatos de FIFA/Game.  
- `style.css` → estilos compartilhados e responsivos.  
- `script.js` → lógica em JavaScript (embutida em cada página).

## 📱 Como Usar
1. Abra o projeto pelo link: [Sorteio FC](https://sorteio-fc.netlify.app/).  
2. Escolha a seção desejada (Futebol Real ou FIFA/Game).  
3. Insira os nomes dos jogadores (e times, no caso do FIFA).  
4. Configure o sorteio (quantidade de equipes ou jogadores por equipe).  
5. Clique em **Sortear** e veja os resultados exibidos na tela.  

---

## 🎯 Objetivo
Facilitar a organização de partidas de futebol e campeonatos de videogame, evitando discussões sobre quem joga em qual time e tornando o processo rápido, justo e divertido.
